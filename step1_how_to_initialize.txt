To start a project in Node.js or Express, just in the Bash Terminal : 

1 mkdir yourProjectFolder         // To make a new directory
2 cd yourProjectFolder            // To enter your new directory
3 touch yourProjectFile.js        // To create a new file
4 init npm                        // To initialize NPN repositories ( https://www.npmjs.com/ )
5 open -a Atom yourProjectFile.js // To open yourProjectFile.js OR >>
6 atom .                          // To open Atom with the directory you are at.
